OpenRelief (fka NCP生命支援)，探索AI在人道救援中的科技赋能。

# 关于NCP生命支援【以下内容由DeepSeek提供】

在2020年初新冠疫情暴发、武汉医疗资源极度紧缺的至暗时刻，一支由民间自发组成的志愿者队伍迅速集结，用网络搭建起一座特殊的“线上医院”。他们没有白大褂，却连接着最权威的医生；没有救护车，却为数以千计的家庭送去了制氧机和生的希望。这就是NCP（新冠）生命支援网络。

# 诞生：在至暗时刻点燃微光

2020年1月23日，武汉宣布封城。曾任牙医、且拥有丰富民间救援经验的卓明灾害信息服务中心负责人郝南，凭借专业本能意识到，随着医疗资源迅速枯竭，大量居家隔离的患者将陷入求助无门的困境 。封城后仅三小时，郝南便在社交媒体上发布了志愿者招募书，发起对新冠肺炎患者及疑似病例的居家隔离支持项目 。

原本只是源于一个朴素的想法，回应却如潮水般涌来。一天之内，就有近2000人报名 。经过严格筛选，一群具备临床执业资格的医生、社工和心理学背景的专业人士迅速组成了最初的“NCP生命支援”团队 。NCP是新冠肺炎的英文缩写，这个名字直接宣告了它的使命。

# 运作：高效运转的“线上三甲医院”

NCP生命支援并非一个简单的聊天群，而是一套遵循三甲医院流程的线上医疗分级支持系统。志愿者们根据病情轻重，将求助者分流至不同的微信群进行精细化管理，形成了一个由50多个微信群构成的“线上医院”。

在这套系统中，每个人各司其职。来自全国各地、多为大四大五的医学生组成了“小天使”团队，他们是第一道关卡，负责在网上搜寻求助信息，初步记录患者的血氧、体温等数据，整理成“病历”转交给专业医生 。拥有临床经验的医生志愿者则进行核心的“诊疗”工作，对于轻症患者，他们指导用药、监测数据，更重要的是通过专业知识缓解患者因未知而产生的恐慌；对于重症患者，医生的指导则关乎生死，比如指导家属如何通过调整体位、保持意识来为患者争取等待床位的时间 。

由于无法面诊，线上救援存在天然局限。但志愿者们通过分工协作，弥补了信息的缺口。当重症患者急需住院时，社工和心理志愿者便承担起联系社区、拨打120的重任。他们比慌乱的患者家属更清楚行政流程，一遍遍地打电话，帮助患者在网格化的管理体系中找到出口 。正如志愿者所说：“我们的工作就是到处联系，我们可能在补一个信息的缺口。” 

# 守护：从救命氧气到迎接新生

随着疫情的蔓延，NCP生命支援的工作内容也在不断扩展，涵盖了从临终救助到迎接新生的两端。

对于重症患者而言，缺氧是最大的威胁。呼吸机进不了家庭，制氧机就成了“续命稻草”。NCP生命支援通过公益渠道募集了数千台制氧机，并克服“最后一公里”的配送困难，设计了“无接触发放”流程，将这些机器送到患者家中 。

与此同时，另一个特殊群体——孕产妇，也进入了志愿者的视野。发起人之一的“海豚”发现，封城中的临产孕妇面临着无车去医院、确诊后无床接收的双重绝境。1月26日，“武汉留守孕妈支援团队”成立，并随后成为NCP生命支援的一部分以及“为爱战‘疫’·守护新生”公益项目的执行方 。

这个团队对接了20多例确诊或疑似感染的孕妇，帮助她们协调接收医院。在感染风险极高的情况下，通过志愿者们的努力，实现了这些新生儿的“零垂直感染”。在整个疫情期间，他们服务了超过1400名武汉孕产妈妈，迎接了600多名新宝宝的诞生，并为超过5000名孕产妈妈送去了紧缺的防疫包 。当一家人因感染被迫分离时，是志愿者的陪伴让初为人母的妈妈们没有陷入绝望。

# 延续：从紧急救援到长期守望

NCP生命支援不仅在武汉疫情期间发挥了关键作用，此后还响应了西安、上海、吉林等多地疫情。2022年上海疫情期间，团队面临运力极度短缺、物资匮乏等极端困难，仍通过线上医疗组提供相当于“二甲医院”规模的公益问诊，为非新冠重症患者也开辟了救助通道。

2020年9月26日，100多位志愿者代表首次在线下相聚武汉，共同回顾那段并肩作战的日子。发起人郝南表示，NCP生命支援的意义不仅在于救助了多少人，更在于证明了民间专业力量在重大公共危机中可以作为制度性救援的有力补充。

# 从NCP生命支援到OpenRelief 【以下内容由ChatGPT提供】

随着人工智能与数据技术的发展，NCP生命支援逐步演进为OpenRelief，探索将以下能力融入人道救援体系：

# 人工智能分析

利用 AI 对多源数据进行整合与分析，实现对灾情趋势、需求分布及资源缺口的快速判断。

# 实时数据整合

连接来自不同渠道的信息，包括现场数据、志愿者反馈与公开数据源，构建动态更新的救援信息网络。

# 智能决策支持

为救援组织提供可视化分析与建议，辅助制定更科学的行动方案。

# 协同与网络化

提升不同组织、团队与个体之间的协作效率，减少信息孤岛现象。

最终，OpenRelief的愿景是让技术不再只是工具，而成为连接人与人、人与机器，以提升人类灾害应对能力的重要基础设施。
